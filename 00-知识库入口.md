# Obsidian AI 记忆库

这是给 Codex、Claude、Cursor、OpenCode 等 agent 共用的本地知识库。

## 已安装组件

- MCPVault `0.12.1`: 把本 Obsidian vault 暴露为标准 MCP 工具。
- projectmem `0.1.6`: 项目主动记忆、失败尝试记录、跨项目全局经验。

## 常用入口

- [[01-Projects/项目索引]]
- [[Agent-Configs/MCP 配置总览]]
- [[Templates/项目记录模板]]

## 接入新项目

在任意项目目录运行：

```bash
/Users/xiaosong/Documents/Obsidian/Scripts/connect-project-memory.sh .
```

它会初始化项目的 `.projectmem/`，并在本知识库中创建对应项目笔记。
