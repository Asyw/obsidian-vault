# pyCapCut — Python自动生成CapCut草稿 ⭐574

> GitHub: https://github.com/GuanYixuan/pyCapCut

## 概述

轻量级、灵活易用的 Python 工具，用于生成和导出 CapCut（剪映）草稿，构建全自动视频剪辑管线。

## 核心功能

- Python 生成 CapCut 草稿文件
- 自动拼接视频片段
- 添加字幕、音乐、特效
- 批量导出草稿
- 可接入自动化工作流

## 适用场景

- 批量短视频生产自动化
- Python 驱动的视频剪辑管线
- 对接 AI 内容生成→自动出片
