## 概述

## 1. 好莱坞赛车电影风格（勒芒风）

适合：品牌大片、运动产品广告

```
风格：好莱坞专业赛车电影（勒芒风格），电影感夜晚，雨天，高风险运动。
时长：15秒。

[00-05s] 老将（内景/特写）
雨水拍打着赛车上高科技赛车的挡风玻璃。老赛车手（戴头盔）目光冷静而专注。
仪表盘的灯光反射在他的面罩上。他微微点头，低声说："出发吧。"

[05-10s] 挑战者（内景/特写）
切换到旁边对手的车内。年轻的车手紧握方向盘，呼吸沉重，眼神中充满肾上腺素。
他对自己小声说："专注。"

[10-15s] 绿灯亮起（远景动作）
起跑灯变绿。两辆车在潮湿的沥青路面上同步加速。水花飞溅到镜头上。
运动模糊将体育场的灯光变成长长的彩色线条。
```

## 2. 王家卫电影风格（雨夜电话亭）

适合：文艺短片、情感故事、品牌情感宣传

```
[电影风格]: 90年代香港艺术电影风格，复古胶片感，高ISO颗粒，
暧昧的黄绿色调，抽帧效果，忧郁氛围。

[核心对白]: "如果记忆是一个罐头，我希望它永远不会过期。"

[视频时长]: 10 秒

[00:00-00:04] 镜头1：透过玻璃窥视
红色公共电话亭，穿风衣的人紧握听筒没有说话。雨水流过玻璃，
像油画一样扭曲了脸。

[00:04-00:07] 镜头2：极端特写
嘴唇轻微颤抖，欲言又止。街道霓虹灯虚化光点流过脸上，忽明忽暗。

[00:07-00:10] 镜头3：慢速快门拖影
挂断电话，走入雨中人群。抽帧效果，背影模糊带拖影，
背景是流动的城市车灯形成的光轨。

[技术参数]: 模拟手持摄影机，浅景深，偏色
```

【角色】

【全局要求】9:16 竖屏，电影质感，奢华宴会厅，光影高级。

## 7. 直播带货口播 — SK-II

适合：直播切片、带货视频

```
生成一段 15 秒带货口播视频，产品为图中面霜。要求人物亲手涂手上，
给出近景展示，侧脸转动展示细节，镜头从半身到特写切换 3 次。
口播要包含卖点，最后给出强行动号召。自动生成字幕，节奏紧凑，
带轻微环境音。
```

**特点**：AI 自动识别产品，匹配对应口播台词（成分、功效），口型同步。

## 9. AI漫剧 — 武侠打斗

适合：小说推文、动态漫画

```
帮我根据下面的小说剧情要求生成动漫，将画风动作打斗表现严格对齐
@视频1 的风格。

老者身形如鹤，袖袍一抖便卷起劲风，逼得年轻剑客连退三步。
后者骤然低喝，剑尖挽出七朵虚影，正是"七星照月"的杀招。
老者却似早有所料，枯瘦五指如鹰爪扣住剑身，内力一震—
长剑寸寸崩断！年轻人弃剑疾退，反手抽出腰间软鞭，
鞭梢如毒蛇吐信直取老者双目
```

**技巧**：先找一段参考画风视频，让 Seedance 对齐风格。

## 10. 动态海报 — 九宫格分镜

适合：品牌宣发、活动预热

```
一个15秒竖屏9:16的卡点音乐短片，背景音乐使用@音频1，
视频全程保持固定机位景别不变，视频中的元素陆续按@图片1中的
九宫格分镜依次显现，从左上角第一个镜头作为开始从左到右、
从上到下依次推进。
```

**流程**：
1. 先用作图工具（如 NanoBanana Pro）做九宫格分镜图
2. 让 Seedance 2.0 按图依次展现元素
3. Tips：可以在提示词中明确告诉它元素出现的顺序

## 附：提示词技巧总结

| 技巧 | 说明 |
|------|------|
| 分镜头时间码 | `[00-05s] 镜头1：...` 精确控制每一段 |
| 四要素模板 | `[风格]` + `[角色]` + `[全局要求]` + `[分镜]` |
| @引用 | 用 `@图1` `@视频1` `@音频1` 引用素材 |
| 人物一致性 | 明确描述服装、妆容、发型 |
| 参考对齐 | 用 `严格对齐@视频1 的风格` 锁定风格 |
