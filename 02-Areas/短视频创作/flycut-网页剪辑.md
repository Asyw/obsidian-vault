# flycut — 网页端视频编辑工具 ⭐923

> GitHub: https://github.com/x007xyz/flycut

## 概述

基于 WebCodecs 实现的网页端视频编辑工具，类似剪映 Web 版。无需安装客户端，浏览器即可完成视频剪辑。

## 核心功能

- 网页端视频剪辑（WebCodecs）
- 类似剪映的交互体验
- 无需安装，浏览器运行
- 支持基本剪辑操作

## 适用场景

- 不想安装软件的轻量剪辑需求
- 网页端集成视频编辑功能
- 短视频快速处理
