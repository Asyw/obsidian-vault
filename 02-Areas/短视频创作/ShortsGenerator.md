# ShortsGenerator — 本地Shorts自动生成 ⭐324

> GitHub: https://github.com/leamsigc/ShortsGenerator

## 概述

在本地自动生成 YouTube Shorts 内容工具，几步简单操作即可批量生产短视频。

## 核心功能

- 本地运行，无需云端 API
- 自动化 Shorts 生成
- 简单配置即可使用
- 支持多种内容来源

## 适用场景

- YouTube Shorts 频道自动化运营
- 个人创作者本地自动化生产
- 无需付费 API 的短视频生成
