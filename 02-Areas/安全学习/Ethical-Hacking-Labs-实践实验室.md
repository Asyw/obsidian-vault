# Ethical-Hacking-Labs — CEH 实战实验室 ⭐3.8K

> GitHub: https://github.com/Samsar4/Ethical-Hacking-Labs

## 概述

基于 CEH（认证伦理黑客）体系的实践教程，侧重动手操作。适合伦理黑客、网络安全学生和系统管理员。

## 前置要求

- 至少 8GB 内存（推荐 16GB）
- 至少 80GB 磁盘空间
- 硬件虚拟化支持

## 模块结构

### Module 0: 核心知识
- Networking 101 / Lab Building / Linux 入门 / 脚本 / 信息安全 101

### Module 1-4: 信息收集
- 踩点与侦察（Maltego, Recon-ng, Metasploit, theHarvester）
- 网络扫描（Nmap, hping3, TTL 检测）
- 枚举（Nmap 枚举, SNMP, Enum4Linux）
- 漏洞分析

### Module 5-8: 攻击技术
- 系统入侵（LLMNR 欺骗、SAM 哈希、提权、隐写术、NTFS 流）
- 恶意软件（njRAT, HTTP Trojan, 混淆、恶意软件分析）
- 嗅探
- 社会工程学

### Module 9-11: 进阶
- DoS 攻击
- 会话劫持
- Bonus 内容
