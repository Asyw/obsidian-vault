## 概述

渗透测试学习路线 + 按分类整理的黑客工具、资源和参考。

## 核心内容

### 渗透测试基本步骤

```
情报收集 → 扫描 → 获取访问 → 维持访问 → 清除痕迹 → 报告
```

### 工具分类

| 分类 | 代表工具 |
|------|---------|
| 信息收集 | theHarvester, Nmap, Sn1per, PhoneInfoga |
| 密码攻击 | John the Ripper, hashcat, Hydra |
| 无线测试 | Aircrack-ng, bettercap, Airgeddon |
| 漏洞利用 | SQLmap, XSStrike, Nuclei, Commix |
| 嗅探与欺骗 | Wireshark, bettercap |
| Web 攻击 | WPScan, Droopescan |
| 后利用 | Metasploit, TheFatRat |
| 框架 | Metasploit, Beef, MobSF, Burp Suite |
