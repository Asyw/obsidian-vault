# h4cker — 网络安全资源大全 ⭐28K

> 维护者：Omar Santos (@santosomar)

## 仓库信息
- **GitHub**: https://github.com/The-Art-of-Hacking/h4cker
- **授权**: MIT

## 概述

综合性网络安全参考库，涵盖攻击安全、防御安全、云和容器安全、应用安全、AI 安全、认证和实验室资源。

## 目录结构

| 目录 | 内容 |
|------|------|
| `cybersecurity-domains/` | 领域分类：基础、攻击、防御、应用安全、云安全、密码学、硬件安全、GRC |
| `ai/` | AI 安全、LLM 工程、AI 辅助事件响应 |
| `certifications/` | 认证路线图（CISSP、OSCP、CCNA、云认证等） |
| `build-your-own-lab/` | 实验室搭建和安全靶场资源 |
| `training-reference/` | Cheat Sheet、O'Reilly 资源、工具索引 |

## 适用人群

- 安全初学者和专业人士
- 认证备考者
- 想搭建个人实验室的学习者
