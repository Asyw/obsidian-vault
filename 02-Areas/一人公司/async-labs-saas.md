# async-labs-saas — React SaaS样板 ⭐4.4K

> GitHub: https://github.com/async-labs/saas

用 React + Material-UI + Next.js + MobX + Express + Mongoose 构建 SaaS 业务的样板项目。

## 技术栈
- React + Material-UI
- Next.js
- MobX 状态管理
- Express 后端
- MongoDB/Mongoose
- WebSockets

## 适用场景
需要实时功能（WebSocket）的一人公司 SaaS 产品。
