# SaaS-Boilerplate — Next.js SaaS样板 ⭐7.2K

> GitHub: https://github.com/ixartz/SaaS-Boilerplate

## 概述

Next.js + Tailwind CSS + Shadcn UI + TypeScript 构建的 SaaS 样板项目。全栈 React 应用，包含认证、数据库、邮件、SEO。

## 核心功能
- Next.js 15 App Router
- Tailwind CSS + Shadcn UI
- TypeScript
- 认证系统
- 数据库集成
- 邮件发送
- SEO 优化
- Stripe 支付集成

## 适用场景
一人公司快速启动 SaaS 产品，省去从零搭建的时间。
