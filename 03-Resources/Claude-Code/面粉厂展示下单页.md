# 面粉厂展示下单页

> 文件：`demo_shop.html`（单文件，~360KB，图片 base64 内嵌）
> 技术栈：纯 HTML + CSS + Vanilla JS，零外部依赖

## 页面结构

```
.lang-bar          → 语言切换栏（中文/维吾尔语）
.hero              → 头部横幅（渐变背景 + 标题 + 标语 + 地址）
.story             → 品牌故事区
.prod-grid         → 产品卡片网格（grid 5列）
  .prod-card       → 单个产品卡片（图片上 + 信息下）
.order-sec         → 下单表单
.footer            → 页脚
```

## CSS 颜色系统

- **暖棕色**: `#3d2e1e` 主文字, `#5a3e1b` 标题
- **绿色**: `#3d6b2e` 深绿, `#5a8a3c` 按钮, `#7a9e4a` 渐变
- **金色**: `#d4a84b` / `#c4963a` / `#b88a30` 高亮
- **背景**: `#faf5eb` 页, `#fff` 卡片
- **橙**: `#b85c2e` 价格

## 双语切换模式

```js
var curLang = "zh";
function swLang(l) {
  document.body.className = l === "ug" ? "ug" : "";
  // 更新 active 状态 + 表单 placeholder + 产品下拉
}
```
CSS: `body.ug [lang="zh"]{display:none}` / `body.ug [lang="ug"]{display:inline}`

## 可复用组件
- 双语布局: CSS class + `[lang]` 属性选择器 + `body.ug` 切换
- 产品卡片: `.prod-grid` (grid 5列) + `.prod-card` flex 竖排
- 简单表单: `.order-sec > .fg` + `.submit-btn`
- 本地持久化: `localStorage` 存 JSON 数组
