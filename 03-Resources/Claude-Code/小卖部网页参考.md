# 小卖部网页参考

> 项目路径：`C:\Users\NINGMEI\Desktop\AI\小卖部网页\shop.html`

## 项目概述

- **类型**: 小卖部/便利店商品展示 + 购物车 + 微信下单页
- **技术栈**: 纯 HTML + CSS + Vanilla JS，零外部依赖
- **商品**: 11个，分4类（面粉、调味料、干货、日用品）
- **图片**: base64 内嵌

## CSS 关键模式

同面粉厂项目的**暖棕/绿/金色系**。

### 商品网格（flex-wrap 2列，桌面端3列）
```css
.prod-grid { display: flex; flex-wrap: wrap; gap: 12px; padding: 14px; }
.prod-card {
  background: #fff; border-radius: 14px;
  box-shadow: 0 2px 12px rgba(0,0,0,0.05);
  width: calc(50% - 6px); display: flex; flex-direction: column;
}
@media(min-width:600px) {
  .prod-card { width: calc(33.33% - 7px); }
}
```

### 购物车（底部固定毛玻璃效果）
- fixed bottom，点击展开 `.cart-items`（max-height 过渡动画）
- 增减数量/删除/显示总价

### 分类筛选
- sticky 分类按钮栏（全部、面粉、调味料、干货、日用品）
- 点击切换 `.active`，JS 控制显隐
