# 短视频制作工作流

## 使用方式

用户以后可以直接给 Codex：

```text
剧本或提示词
视频比例：9:16 或 16:9
可选：时长、风格、是否数字人出镜、是否要字幕/封面
```

Codex 按请求生成短视频，不默认限制为数字人口播。

## 比例

```text
9:16 -> 1080x1920，适合抖音/视频号/小红书竖屏短视频
16:9 -> 1920x1080，适合 B站/YouTube/横屏课程或纪录片感视频
1:1 -> 1080x1080，备用方形信息流
```

如果用户没有指定比例，默认读取：

```text
SHORT_VIDEO_ASPECT_RATIO
```

## 默认管线

```text
输入剧本或提示词
-> 分析目标比例和风格
-> 改写成短视频脚本
-> 拆成分镜
-> 用复刻声音生成旁白
-> 生成画面素材或视频片段
-> 生成字幕
-> 生成封面
-> 合成 MP4
-> 保存到 outputs/videos
```

## 模式

```text
hybrid: 复刻声音配音 + AI画面/视频 + 字幕封面剪辑
talking_head: 数字人形象出镜口播
visual_story: 不出镜，剧情/知识/历史画面配旁白
image_to_video: 图片素材转动态视频
```

默认模式：

```text
SHORT_VIDEO_MODE=hybrid
```

只有 `talking_head` 模式必须使用数字人形象 ID。
