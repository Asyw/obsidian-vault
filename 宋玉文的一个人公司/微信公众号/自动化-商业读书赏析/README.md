# 微信公众号商业读书赏析自动化

这个文件夹用于把一本商业相关书籍，快速整理成适合微信公众号发布的读书赏析文章。

## 使用顺序

1. 打开 `01-书籍输入卡片模板.md`，复制一份，填入书名、作者、阅读感受、核心观点和现实联想。
2. 把填写好的内容贴进 `02-公众号文章生成提示词.md` 的提示词中，让 AI 生成初稿。
3. 用 `04-公众号成稿结构模板.md` 检查文章结构是否完整。
4. 根据 `03-图片搭配与封面提示词.md` 给文章补充封面图、段落配图和朋友圈转发文案。
5. 参考 `示例-创新者的窘境.md` 的成稿质感，判断是否符合你的公众号风格。

## 建议定位

不要把文章写成普通书摘，而是写成：

> 一本书 + 一个商业问题 + 你的个人判断

每篇文章最好回答一个清晰问题，例如：

- 为什么大公司总是错过新机会？
- 小公司为什么不能一开始就追求大而全？
- AI 时代，什么能力会变得更值钱？
- 创业者应该如何判断一个市场是否值得进入？

## 产出清单

每本书建议最终产出：

- 1 篇公众号正文
- 3 个可选标题
- 1 段摘要
- 1 条朋友圈转发文案
- 1 个封面图提示词
- 2-4 个正文配图建议
- 3-5 个适合做海报的金句

## 发布自动化

默认发布流程先创建微信公众号草稿，人工检查排版后再发布。

1. 复制 `.env.example` 为 `.env`，填写公众号后台里的 `WECHAT_APP_ID` 和 `WECHAT_APP_SECRET`。
2. 确认公众号后台已把当前机器出口 IP 加入开发者 IP 白名单。
3. 安装依赖：`python3 -m pip install requests`
4. 创建草稿：

```bash
python3 scripts/wechat_publish.py 示例-创新者的窘境.md --env .env
```

如果要手动指定微信后台里的标题：

```bash
python3 scripts/wechat_publish.py 示例-创新者的窘境.md --env .env --title "创新者的窘境"
```

如已确认文章可直接发布，再加 `--publish`：

```bash
python3 scripts/wechat_publish.py 示例-创新者的窘境.md --env .env --publish
```

查询发布状态：

```bash
python3 scripts/wechat_publish.py 示例-创新者的窘境.md --env .env --status publish_id
```
