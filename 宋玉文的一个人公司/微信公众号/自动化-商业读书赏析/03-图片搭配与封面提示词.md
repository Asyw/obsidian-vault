# 图片搭配与封面提示词

图片的作用不是装饰，而是帮助读者更快进入商业问题。

## 推荐图片类型

### 1. 封面图

适合用来强化文章主题，而不是简单放书封面。

可选方向：

- 一本打开的商业书，旁边有咖啡、笔记、增长曲线、行业报告。
- 城市办公室夜景，桌上有书和标注过的商业图表。
- 创业者或管理者在白板前思考业务变化。
- 抽象但克制的商业竞争画面，例如小公司绕开巨头、旧行业被新技术重塑。

封面图提示词模板：

```text
一张适合微信公众号商业读书文章的封面图，主题是【文章核心问题】。画面包含一本打开的商业书、手写笔记、简洁的商业图表和城市办公室背景。风格真实、克制、现代、有思考感，适合商业读者，不要卡通，不要夸张文字，不要过度科技感，横版构图，留出标题排版空间。
```

### 2. 段落配图

每篇文章建议 2-4 张：

- 第 1 张：开头问题图，用来制造阅读场景。
- 第 2 张：核心观点图，用来解释书里的关键逻辑。
- 第 3 张：行业启发图，用来连接当下商业。
- 第 4 张：阅读建议图，用来营造笔记和复盘感。

## 图片插入位置

建议插在这些位置：

1. 标题后、正文前：封面图。
2. “这本书主要讲了什么”前：书籍/笔记场景图。
3. “对当下商业有什么启发”前：行业变化或商业场景图。
4. “我建议你如何阅读”前：阅读方法/笔记图。

## 常用图片提示词

### 商业思考型

```text
真实摄影风格，一张办公桌上的商业读书笔记，打开的书、手写批注、笔记本、钢笔、简洁的增长曲线图表，窗外是现代城市办公楼，光线自然，画面干净克制，适合微信公众号商业文章配图。
```

### 行业变化型

```text
真实摄影风格，现代企业会议室里，一块白板上画着旧业务到新业务的转型路径，桌上有笔记本电脑、行业报告和便利贴，画面体现商业变化、创新压力和战略选择，克制、专业、现代。
```

### 个人收获型

```text
真实摄影风格，一个人在安静的书桌前阅读商业书籍并做笔记，旁边有电脑、便签和简单图表，氛围专注、温暖、理性，适合表达读书后的个人思考和商业复盘。
```

## 注意事项

- 不建议直接使用未经授权的书籍封面作为公众号主图。
- 图片最好服务于观点，不要只追求好看。
- 商业读书文章适合克制、真实、干净的视觉风格。
- 如果文章谈 AI，可以有技术元素，但不要做成满屏蓝紫色科技风。
